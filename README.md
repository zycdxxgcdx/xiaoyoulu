# xiaoyoulu
校友录B/S架构
# 技术栈
前端: elementui + vue
后端: sprintboot + mybatis + websocket
富文本编辑器： wangEditor
# 项目启动
前端：
cd到final_project文件夹
终端里输入 npm run serve
